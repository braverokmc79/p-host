DDoS Proxy Service



  • 빠른 장애 조치로 안정성 향상

    Load Balancing은 장애 시간을 가장 가용성과 응답성이 높은 서버를 동적 분산하여 서비스의 가동 시간을 높이는 데 도움을 줍니다.

  • 보안 비용 절감

    IDC에 입주되어 있는 고성능. 고가의 서버를 추가 임대할 필요 없이 필요한 DDoS 보안 서비스만 받으실 수 있습니다.

  • 대용량 DDoS 보안

    197 Tbps 이상의 네트워크 완화 용량, 즉각적인 위협 감지, 대부분의 위협의 경우 3초 미만의 완화 시간(TTM) 등을 통해 응용 프로그램을 가장 정교한 멀티 벡터 DDoS 공격으로부터 보호하고 프록시를 사용합니다.

  • 익명성

    외부에 클라우드 플레어 아이피를 사용하여 익명으로 서버를 운용할 수 있다. 프록시 서버를 통해 한 단계의 보안을 더 할 수 있기 때문에 컴퓨터 보안을 유지할 수 있습니다.

  • 실시간 트래픽 가속

    Argo Smart Routing과 통합하여 TCP 트래픽을 더 빨리 전송합니다. 해당 서비스는 수백만 개의 인터넷 자산에 흐르는 트래픽을 학습하여 ML 기반 지능형 라우팅을 실행하고 실시간으로 네트워크 정체를 피합니다.

  • 편의성

    서버의 직접적인 이동이 없이, DDoS 방어가 가능할 수 있게 하는 서비스입니다. 자사 IDC에 서버가 설치되어 있지 않더라도, 우회 방어 서비스로 이용하실 수 있습니다.

왜, 우회 방어 솔루션을 사용할까요?


기존 시스템 구성 변경이 어려운 분들

기존에 구성되어 있는 복잡한 시스템을 이전하기 어렵거나 시간이 없어서 작업하지 못하시는 분들이 사용합니다.

DDoS 공격으로 인한 서비스 장애로 빠른 복구가 필요한 분들

현재 DDoS 공격으로 인해 서비스 장애를 받고 있으며 빠른 정상화 처리가 필요하고 저렴한 비용으로 보안이 필요하신 분들이 사용합니다.

일시적으로 DDoS 방어가 필요한 분들

월 단위 가 아닌 7일, 15일 등 일시적으로 DDoS 보안이 필요한 분들이 우회 보안 서비스를 사용합니다.
